# CLI-To-do-list-using-Rust

## 1. What is Git?
Git is an open source distributed version control software that is designed to track changes in files and code during software development and enable efficient and seamless collaboration and development between different developers.

## 2. How to push code to GitHub?
i) Make sure you are in a Git repository, and open the terminal path in the folder of the repo.  
ii) Create an empty GitHub repository if not yet linked, and connect it to the local repo using -  
  **git remote add origin <REMOTE_URL>**  
iii) Once linked, stage the files for commit using -  
  **git add <filename_1>, <filename_2>, ...**  
  or **git add .** for multiple files  
iv) Commit the changes with the commit message using -  
  **git commit -m "<commit_msg>"**  
v) Push the changes to the GitHub repository using -  
  **git push origin <main/master>** depending on the branch name  

## 3. What is markdown?
Markdown is a lightweight markup language that supports text formatting using symbols like * or #, making it very easy to read, write, and display, and hence versatile.

## 4. Markdown Syntax?
i) # for headings 1  
ii) ## for heading 2  
iii) ### for heading 3, and so on...  
iv) Start a new paragraph by leaving a blank line.  
v) End a line with multiple spaces for line breaks.  
vi) Use single asterisks * for *italic* text.  
vii) Use double asterisks ** for **bold** text.  
viii) Use triple asterisks *** for ***italic and bold*** text.  
ix) Use double tidles for ~~strikethrough~~ text.  
x) Use -, +, or * at the start of a line for unordered lists.  
xi) Use numbering like 1., 2., 3. ... for ordered lists.
